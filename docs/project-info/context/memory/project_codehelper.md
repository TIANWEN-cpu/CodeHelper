项目名称：CodeHelper（AI驱动的本地编程刷题+IDE+个性化学习系统）
项目路径：D:\codehelper
用户环境：Windows 11, 使用 Claude Code CLI
交付形式：打包为单个 .exe 可执行文件（electron-builder）
界面语言：中文
AI服务：用户已有 OpenAI 格式 API Key
用户角色：个人学习工具，单用户，无需多用户/权限系统

**Why:** 用户希望AI独立完成全部开发，自己只体验最终产品
**How to apply:** 所有技术决策直接采用最优方案，不需要询问技术偏好；最终需要能打包为.exe
